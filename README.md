# 🎓 University Bot - بوت الجامعة المتكامل

بوت تليغرام ذكي لإدارة وتنظيم الحياة الجامعية للطلاب والمدرسين.

## ✨ الميزات الرئيسية

### 1️⃣ **فحص العضوية** 🔐
- التحقق الآلي من عضوية المستخدمين في القناة
- السماح بالوصول للميزات فقط للأعضاء المسجلين
- دعم تحديثات العضوية الفورية

### 2️⃣ **إدارة الجدول الدراسي** 📅
- عرض جدول الدراسة اليومي والأسبوعي
- إضافة وحذف المحاضرات (للمشرفين)
- تنظيم حسب الأيام والأوقات
- عرض معلومات الأستاذ والقاعة

### 3️⃣ **تذكيرات الدراسة** ⏰
- تذكيرات يومية للمحاضرات
- تنبيهات الامتحانات القادمة
- تذكيرات المواعيد النهائية
- إمكانية تخصيص التذكيرات
